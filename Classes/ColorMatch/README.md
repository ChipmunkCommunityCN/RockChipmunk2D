# 说明

-  本DEMO范例基于Chipmunk作者slembcke的项目：[https://github.com/slembcke/ChipmunkColorMatch](https://github.com/slembcke/ChipmunkColorMatch) 

# 游戏玩法

-  点击任意球会消除掉，4个以上颜色相同的球接触会消除

# 涉及算法

-  并查集算法
